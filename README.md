# PHP-Smart-Crop
A GD2 method for smart resizing and cropping, based on entropy and color of the image.

Overhaul of [gschoppe/PHP-Smart-Crop](https://github.com/gschoppe/PHP-Smart-Crop) : 
* PSR-4
* namespace
* autoload
* phpunit tests
* composer

## Install
With composer :
```
$ composer require trismegiste/smart-crop
